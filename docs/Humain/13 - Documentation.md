# Documentation Kerpta (kerpta.fr/docs)

La documentation de Kerpta est accessible directement sur `kerpta.fr/docs`. Elle couvre l'installation, l'utilisation de toutes les fonctionnalités, la référence API et les informations pour les contributeurs open source.

---

## Sections disponibles

### Guide d'installation (kerpta.fr/docs/installation)

Pour les entreprises ou développeurs qui souhaitent héberger Kerpta sur leur propre serveur. Vous y trouverez les prérequis techniques, le guide pas-à-pas avec le script d'installation automatique (`install.sh`), les instructions de déploiement sur un VPS, la configuration des variables d'environnement, et la procédure de mise à jour.

### Guide utilisateur (kerpta.fr/docs/guide)

Explications fonctionnelles pour chaque module : comment créer son organisation, ajouter un client, rédiger un devis, émettre une facture, gérer les achats fournisseurs, soumettre une note de frais, générer des fiches de paie, faire son rapprochement bancaire, signer un document, gérer son mini-site vitrine, etc.

### Référence API (kerpta.fr/docs/api)

Documentation technique de tous les endpoints de l'API REST Kerpta — générée automatiquement depuis le code source, toujours à jour.

### Guide des contributeurs (kerpta.fr/docs/contribuer)

Pour les développeurs qui souhaitent contribuer au projet open source : architecture technique, installation de l'environnement de développement, conventions de code, workflow des Pull Requests, et roadmap publique des fonctionnalités prévues.

---

## Pourquoi une documentation intégrée ?

La documentation est hébergée directement sur kerpta.fr, sans outil externe. Elle est versionée dans le même dépôt Git que le code — quand une fonctionnalité évolue, la doc est mise à jour en même temps. Résultat : la documentation est toujours synchronisée avec la version en ligne.

Une barre de recherche permet de trouver rapidement n'importe quel sujet. La navigation est adaptée aux mobiles.
